多语言

GitBook支持使用多语言来构建书本。按照GitBook的标准格式，每个语言应该作为一个子目录，命名为 LANGS.md 的文件应该遵循下面的格式并出现在仓库的根目录下：

* [英语](en/)
* [法语](fr/)
* [西班牙语](es/)

你可以从 Learn Git 这本书中看到一个完整的例子。
